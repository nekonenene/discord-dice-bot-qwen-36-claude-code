# Discord Dice Bot

Go言語で書かれたDiscordダイスロールボットです。ボットをメンションしてダイス記法を指定すると、指定された個数・面数のダイスを振って結果を返します。

## 使い方

ボットをメンションしてダイス記法を入力します。

```
<@ボットのID> 4D6
```

これにより、6面ダイスを4個振り、結果をEmbedで返します。

### ダイス記法

`NDM` の形式で指定します。

| 記法 | 意味 |
|------|------|
| `4D6` | 6面ダイスを4個振る |
| `2D100` | 100面ダイスを2個振る |
| `1D20` | 20面ダイスを1個振る |

### 制約

- 面数: 1〜100の整数
- 個数: 1〜100の整数
- それ以外の形式で指定された場合はエラーメッセージを返します

## セットアップ

1. Go 1.26以上をインストール

2. 依存関係をダウンロード

```bash
go mod download
```

3. Discord Botトークンを環境変数に設定

```bash
export DISCORD_BOT_TOKEN="your-bot-token-here"
```

4. ボットを起動

```bash
go run main.go
```

## 開発

### コードの整形

```bash
go fmt ./...
```

### ビルド

```bash
go build -o dice-bot main.go
```
